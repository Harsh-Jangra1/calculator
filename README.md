# Calculator

A responsive calculator built with HTML, CSS, and JavaScript. It provides a clean, intuitive interface for performing basic arithmetic operations and works seamlessly across desktop and mobile devices.

## Features

- Basic arithmetic operations
- Responsive design
- Mobile-friendly interface
- Clean and intuitive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Live Demo

https://calculator-mu-six-28.vercel.app/

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Harsh-Jangra1/calculator.git
   ```

2. Open the project folder.

3. Open `index.html` in your browser.
